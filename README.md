# Cockpit BLE


```
Keyboard: Cockpit BLE
Default Layer: QWERTY ENG
Keyboard Firmware: ZMK
Microcontroller: Nordic nRF52840
```

```
Adding zmk studio support for Cockpit BLE keyboard. 
ZMK Studio unlock key combos: Q+W
```
